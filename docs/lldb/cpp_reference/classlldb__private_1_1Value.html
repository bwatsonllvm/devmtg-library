◆ m_value

Scalar lldb_private::Value::m_value
protected

Represents a value, which can be a scalar, a load address, a file address, or a host address.

The interpretation of m_value depends on m_value_type:

  • Scalar: m_value contains the scalar value.
  • Load Address: m_value contains the load address.
  • File Address: m_value contains the file address.
  • Host Address: m_value contains a pointer to the start of the buffer in host memory. Currently, this can point to either:
    • The m_data_buffer of this Value instance (e.g., in DWARF computations).
    • The m_data of a Value Object containing this Value.
