150 /// Extract the `__crash_info` annotations from each of the target's
151 /// modules.
152 ///
153 /// If the platform have a crashed processes with a `__crash_info` section,
154 /// extract the section to gather the messages annotations and the abort
155 /// cause.
156 ///
157 /// \param[in] process
158 /// The crashed process.
159 ///
160 /// \return
161 /// A structured data array containing at each entry in each entry, the
162 /// module spec, its UUID, the crash messages and the abort cause.
163 /// \b nullptr if process has no crash information annotations.
